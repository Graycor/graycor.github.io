Camera blocked — file:// not supported

Android Chrome (and some desktop browsers) block camera access when a file is opened directly from storage. You must serve this file over HTTP.

Easiest fix — Python local server:
Open a terminal in the folder containing this file and run:
python3 -m http.server 8080
Then open http://localhost:8080/book-scanner.html in Chrome.

On Android: run the server on your PC, find your PC's local IP (e.g. 192.168.1.x), then open http://192.168.1.x:8080/book-scanner.html in Chrome on your phone while on the same Wi-Fi.

Or upload once to any free host (GitHub Pages, Netlify Drop) and open the HTTPS URL on any device.
